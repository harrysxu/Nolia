# Nolia Computer Use 桌面验收清单

本文是 `COMPREHENSIVE_TEST_PLAN.md` 的桌面执行版，用于通过 Computer Use 验证安装版或打包版 Nolia。每轮执行应记录测试环境、测试工作区、执行人、执行时间、通过项、失败项、阻塞项和截图/日志位置。

## 1. 测试环境

| 项目 | 要求 |
| --- | --- |
| App | 优先使用当前系统重新打包并安装后的 Nolia；可补充验证 `release/*unpacked*` 目录 |
| 系统 | macOS、Windows 或 Linux，本机中文界面 |
| 数据保护 | 测试前备份当前系统的 Nolia 全局应用数据目录 |
| 测试工作区 | 临时工作区，包含 Markdown、JSON、文本、图片、PDF、音频、视频、压缩包、未知二进制文件 |
| 自动化基线 | `npm run typecheck`、`npm run lint`、`npm test`、`npm run e2e`、`npm run build` 全部通过 |
