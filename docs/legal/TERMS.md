# Nolia 使用条款

生效日期：2026-05-31

本文适用于公开分发的 Nolia 应用、源码、文档和相关项目材料。使用、复制、修改或再分发 Nolia，即表示你同意遵守本文和项目开源许可证。

## 许可

Nolia 应用、源代码、图标、文档和相关资产采用 [项目许可](LICENSE.md) 中说明的 GNU General Public License v3.0 or later 开源许可。

用户创建或导入的笔记、文档、附件和工作区数据归用户所有。

## 使用范围

用户可以在自己的 macOS、Windows 或 Linux 设备上安装和使用 Nolia，用于创建、编辑、整理和预览本地文档。

使用和再分发 Nolia 时，需要遵守 GPLv3 以及适用法律。不得：

- 移除版权、许可或第三方声明。
- 将修改版或二进制分发为违反 GPLv3 条款的专有版本。
- 使用应用从事违法、侵权或破坏性行为。

## 本地数据责任

Nolia 直接读写用户选择的本地工作区。用户应自行负责：

- 选择合适的工作区位置。
- 备份重要数据。
- 处理同步盘、Git、外部编辑器带来的冲突。
- 审慎安装和启用外部插件。

## 插件

外部插件由插件作者负责。启用插件前，用户应检查插件来源、权限和用途。

如果插件具有写入、删除、剪贴板或网络权限，可能影响用户数据安全。

## 免责声明

在法律允许范围内，Nolia 按“现状”提供，不保证无错误、不中断、适合特定用途或不会造成数据丢失。

用户应在重要数据上使用独立备份方案。

## 变更

发布者可以随版本更新调整功能、文档和条款。重大变更会在发布说明、仓库文档或分发页面中说明。
